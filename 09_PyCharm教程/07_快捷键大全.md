"""
【常用组合】

1. 快速导航：
   Ctrl + N → 输入类名 → Enter

2. 快速编辑：
   Alt + Insert → 选择生成内容

3. 快速重构：
   选中代码 → Ctrl + Alt + M → 输入方法名

4. 快速修复：
   Alt + Enter → 选择修复方案

5. 快速搜索：
   Ctrl + Shift + F → 输入搜索内容

6. 快速调试：
   点击行号 → F9 运行 → F8 单步

【记忆技巧】

- Ctrl：基本操作
- Ctrl + Shift：扩展操作（全局、反向等）
- Alt：替代操作（导航、选择等）
- Ctrl + Alt：重构操作
- Shift + F*：功能键的扩展

【推荐设置】

Settings → Keymap：
- 根据习惯选择键位方案
- 可以自定义快捷键
