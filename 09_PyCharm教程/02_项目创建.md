## 1. 创建新项目

### 启动界面创建

1. 打开 PyCharm
2. 点击 `New Project`
3. 选择项目位置和类型
4. 配置 Python 解释器

### 项目类型

| 类型 | 说明 |
|------|------|
| Pure Python | 纯 Python 项目 |
| Flask | Web 框架项目 |
| Django | Web 框架项目 |
| FastAPI | API 框架项目 |

### 推荐设置

```
项目位置：选择合适的目录
解释器：New environment using Virtualenv
继承全局包：可选
```

## 2. 项目结构管理

### 标准项目结构

```
project/
├── src/
│   ├── __init__.py
│   ├── main.py
│   └── utils/
│       └── __init__.py
├── tests/
│   └── test_main.py
├── docs/
├── requirements.txt
├── README.md
└── .gitignore
```

### 标记目录

右键目录 → Mark Directory as：

- **Sources Root**：源代码目录
- **Tests Root**：测试目录
- **Resources Root**：资源目录
- **Excluded**：排除目录

## 3. 解释器配置

### 查看解释器

`File` → `Settings` → `Project` → `Python Interpreter`

### 添加解释器

1. 点击齿轮图标 → `Add`
2. 选择解释器类型：
   - Virtualenv Environment
   - Conda Environment
   - System Interpreter

### 管理包

在解释器设置中可以：
- 查看已安装包
- 安装新包
- 升级/删除包

## 4. 项目设置

### 常用设置

| 设置 | 路径 |

项目配置保存在 `.idea` 目录中。
