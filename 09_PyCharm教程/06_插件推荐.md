## 1. 必备插件

### Chinese (Simplified) Language Pack

中文语言包，将 PyCharm 界面汉化。

### Key Promoter X

显示操作对应的快捷键，帮助记忆快捷键。

### .ignore

支持各种 .ignore 文件格式（.gitignore、.dockerignore 等）。

## 2. 开发增强

### CodeGlance

在编辑器右侧显示代码缩略图，快速导航。

### Rainbow Brackets

彩色括号匹配，便于识别嵌套结构。

### String Manipulation

字符串处理工具：
- 大小写转换
- 驼峰/下划线转换
- 排序、过滤等

### Python Security

Python 安全检查，检测潜在的安全漏洞。

### Kite

AI 代码补全（需要单独安装 Kite 应用）。

### GitHub Copilot

AI 代码助手，自动生成代码建议。

## 5. 如何安装插件

### 安装步骤

1. `File` → `Settings` → `Plugins`
2. 选择 `Marketplace` 标签
3. 搜索插件
4. 点击 `Install`
5. 重启 PyCharm

### 从文件安装

1. `File` → `Settings` → `Plugins`
2. 点击齿轮图标 → `Install Plugin from Disk`
3. 选择插件文件（.zip 或 .jar）

### 禁用/卸载插件

1. `File` → `Settings` → `Plugins`
2. 选择 `Installed` 标签
3. 找到插件，点击禁用或卸载
